Comando: py main.py <gupy_url> <prompt> [-p]

Parâmetros

    gupy_url    Endereço web da vaga na Gupy.

    prompt      Modelo de prompt, pode ser:
                - "a" para obter sugestões para melhoria do currículo ("advice");
                - "f" (padrão) para nível de compatibilidade entre currículo e vaga ("fitness").

    -p          Apenas gera o prompt e finaliza.

Orientações

    - Um arquivo "settings.json" é esperado na pasta raiz com as chaves do arquivo de exemplo.
    Determine o caminho de um arquivo contendo o currículo, uma pasta para salvar os resultados
    e o modelo do Gemini a ser empregado.

    - Prompts são construídos conforme os templates em "files/prompts".

    - Para consultar o Gemini, é necessário definir uma variável de ambiente
    "GOOGLE_API_KEY" com uma chave de acesso.
